Operating System Algorithms


Operating System lies in the category of system software. It basically manages all the resources of the computer. An operating system acts as an interface between the software and different parts of the computer or the computer hardware. The operating system is designed in such a way that it can manage the overall resources and operations of the computer.


A Process Scheduler schedules different processes to be assigned to the CPU based on particular scheduling algorithms. There are many types of scheduling algorithms available, some for CPU scheduling whereas some are for page replacement. Sometimes, it's good to be early


The Shortest Job First or the Shortest Job Next is a C.P.U. scheduling algorithm which has the burst time of a process as its selection criteria.

This simply means that when 2 processes arrive at the same time, the process with the shorter burst time will be executed first.

What if your entire life was just eating and thinking?


The Dining Philosophers' problem is a situation wherein there are 'n' philosophers with the same number of forks on a single table.

This is basically a synchronization problem which is solved by changing the rule for the last philosopher, which is simply the reversal of the order.

How are tracks accessed in a disk?


LOOK is the advanced version of SCAN (elevator) disk scheduling algorithm which gives slightly better seek time than any other algorithm

C-LOOK is an enhanced version of both SCAN as well as LOOK disk scheduling algorithms. This algorithm also uses the idea of wrapping the tracks as a circular cylinder as C-SCAN algorithm.

The oldest should make way for the youngest


In an operating system that uses paging for memory management, a page replacement algorithm is needed to decide which page needs to be replaced when a new page comes in. .

The operating system keeps track of all pages in the memory in a queue, the oldest page being the first one. When a page needs to be replaced page in the front of the queue is selected for removal.
